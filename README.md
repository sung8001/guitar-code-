# 기타 코드 인식 프로그램

## 프로젝트 개요
이 프로젝트는 **실시간으로 기타 연주 코드를 감지**하고, 사용자가 연습하고 있는 코드와 얼마나 일치하는지 시각적으로 피드백을 제공하는 프로그램입니다.  
연주하면서 정확도를 확인할 수 있기 때문에, **효과적으로 기타 연습을 할 수 있는 학습 도구**로 활용할 수 있습니다.

---

## 사용된 라이브러리
- `sounddevice` : 마이크로 실시간 오디오 입력을 받기 위해 사용  
- `numpy` : 오디오 신호 처리 및 RMS 계산, 수치 연산에 사용  
- `librosa` : 오디오에서 주파수와 음계를 추출하고 코드 판정에 사용  
- `tkinter` : GUI를 생성하고 실시간 피드백 화면을 보여주기 위해 사용  
---

## 코드 구성
프로그램은 크게 다음과 같이 구성되어 있습니다:

1. **음계 감지 및 코드 판정**
   - `librosa`를 활용해 오디오에서 주파수를 추출하고, 음계로 변환합니다.
   - 변환된 음계와 목표 코드를 비교하여 정확도를 계산합니다.

2. **GUI 시각화**
   - 원형 게이지: 현재 코드 정확도를 직관적으로 표시합니다.
   - 실시간 그래프: 시간 흐름에 따라 정확도 변화를 기록합니다.

3. **실시간 오디오 루프**
   - `sounddevice`로 오디오를 반복해서 받아 처리합니다.
   - 실시간으로 점수를 계산하고 GUI를 갱신합니다.

4. **사용자 인터페이스**
   - 코드 입력창: 연습할 코드를 입력합니다.
   - Start/Stop 버튼: 실시간 측정 시작/종료
